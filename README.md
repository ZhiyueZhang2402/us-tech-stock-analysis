# US Tech Giants Stock Analysis 📈

A Python-based data analysis project analyzing stock performance of Apple, Microsoft, and Google using WRDS financial data.

## 📊 Project Overview

This project analyzes the stock price performance of three major US tech companies (AAPL, MSFT, GOOGL) from 2023 to 2025 using data from WRDS (Wharton Research Data Services).

### Key Findings

| Stock | Total Return | Annual Volatility | Avg Volume |
|-------|-------------|-------------------|------------|
| GOOGL | 112.31% | 29.15% | 22.46M |
| AAPL | 100.22% | 21.51% | 57.79M |
| MSFT | 75.93% | 22.66% | 24.01M |
